#  PetHealth Lite

Sistema desenvolvido como MVP para a clínica veterinária **PetHealth**, com o objetivo de organizar o fluxo básico de atendimento.

Projeto desenvolvido para a disciplina de Metodologias Ágeis – Técnico em Desenvolvimento de Sistemas - RPV 2026.

---

##  Sobre o Projeto

O PetHealth Lite é uma aplicação web desenvolvida em React que permite:

- Cadastro de pets e seus tutores
- Agendamento de consultas com médico fixo
- Visualização dos pets cadastrados
- Persistência de dados utilizando LocalStorage

O sistema foi planejado utilizando metodologia ágil com organização de backlog, sprints e quadro Kanban no GitHub.

---

##  Funcionalidades

###  Cadastro de Pets
- Cadastro de animal (nome, espécie, idade)
- Cadastro do tutor (nome e telefone)
- Validação com React Hook Form e Zod
- Armazenamento no LocalStorage

###  Agendamento de Consultas
- Seleção de pet cadastrado
- Escolha de data e horário
- Médico fixo definido no sistema
- Salvamento das consultas no LocalStorage

###  Listagem
- Exibição de todos os pets cadastrados
- Visualização completa dos dados
- Atualização automática após cadastro

---

##  Tecnologias Utilizadas

- React
- React Router
- React Hook Form
- Zod
- LocalStorage
- GitHub Projects (Kanban)
- Metodologia Ágil com Sprints

---

## Organização Ágil

O projeto foi organizado em 3 Sprints:

### Sprint 1 – Cadastro de Pacientes
Implementação do cadastro de pets, tutores e persistência de dados.

### Sprint 2 – Fluxo de Consultas
Implementação do agendamento com médico fixo e salvamento das consultas.

### Sprint 3 – Navegação e Dashboard
Criação da Home, listagem e organização do layout com componentização.

Todas as tarefas foram organizadas através de Issues, Milestones e GitHub Projects (Kanban).

---

##  Persistência de Dados

Os dados são armazenados no LocalStorage do navegador, permitindo que as informações não sejam perdidas ao atualizar a página.
